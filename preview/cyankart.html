Case Study

Cyankart.com is a company that sells mobile cases and apparel items online. They were looking to expand their mobile reach and increase conversions, so I was hired to redesign their ecommerce website and provide logistics-related backend solutions. I was required to develop a responsive, cross-platform, and modern Shopify website with search engine optimization and user-engaging patterns.

The most visible change that I made was adding a giant search bar that increased conversions by over 50%. Of course, a huge increase in mobile usage, app downloads, and conversions was noted after I delivered the new website. I also provided Cyankart with analytics logistics management-related backend solutions, which I coded using PHP and integrated with the Shopify admin panel.

The biggest challenge while designing the mobile website for Cyankart was decreasing the load time. I optimized each and every image and minified every part of the code and decreased the usage of additions libraries and instead CSS. The most important part was creating user engagement, so A/B testing on both the navigation bar and products layouts was done to ensure that we delivered the best experience.
